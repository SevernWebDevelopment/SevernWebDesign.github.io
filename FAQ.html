  • What is HTML anyway?

    HTML, in full hypertext markup language, a formatting system for displaying material retrieved over the Internet. ... HTML markup tags specify document elements such as headings, paragraphs, and tables. They mark up a document for display by a computer program known as a Web browser. - Britanicca
